cmd是command的缩写.即命令行 。

mac端命令

目录操作

clear: 清空屏幕的内容
pwd: 查看工作目录
whoami 查看用户名
chown 指定用户操作权限

list show

ls 显示当前目录下的文件内容
ls 可以和路径配合使用，用于显示路径下的内容
ls 后面如果什么都不加，默认显示当前目录下的内容。
ls / 显示跟目录下的内容

cd

cd: 切换工作目录
cd ~ 回到用户目录(刚打开终端时的目录)
cd - 回到上一个目录

/ 根目录
./ 当前目录
../ 上一级目录

创建 make directory

mkdir: 创建目录 mkdir dirname1 dirname2

移动/重命名 remove/rename

mv: 移动/重命名目录 mv dir1 dir2 //dir1 移动文件 dir2 指定文件；仅限同级别目录

删除

rmdir: 删除目录 rmdir dirname1 dirname2

文件操作

touch: 创建一个文件/目录
rm: 删除一个文件
rm -rf: 强制删除所有文件

其他

launchctl 控制OS X系统里的启动进程(launch)

launchctl list
launchctl list | grep mongo
